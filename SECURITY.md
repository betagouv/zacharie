# Politique de sécurité

## Signaler une vulnérabilité

Si vous découvrez une faille de sécurité dans Zacharie, merci de nous la signaler de manière responsable.

**Contact** : [contact@zacharie.beta.gouv.fr](mailto:contact@zacharie.beta.gouv.fr)

Merci de **ne pas** ouvrir d'issue publique pour les vulnérabilités de sécurité.

## Délais

- Accusé de réception sous 3 jours ouvrés.
- Première évaluation sous 10 jours ouvrés.
- Correction déployée dès que possible selon la criticité.

## Périmètre

- Application web : [zacharie.beta.gouv.fr](https://zacharie.beta.gouv.fr)
- API : `api.zacharie.beta.gouv.fr`
- Application mobile (iOS / Android)

## Hors périmètre

- Attaques par déni de service (DoS/DDoS)
- Ingénierie sociale
- Vulnérabilités sur des dépendances tierces sans impact démontré
